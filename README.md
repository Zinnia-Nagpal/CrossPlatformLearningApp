# CrossPlatformLearningApp

This is a learning project built to explore cross-platform application development using .NET MAUI.

## Purpose
The goal of this project is to understand:
- How .NET MAUI enables a single codebase for multiple platforms
- UI development using XAML
- Event handling and navigation between pages
- Project structure and development workflow in Visual Studio

## Current Features
- Default MAUI template application
- Cross-platform support (Windows, Android, iOS)

## Tech Stack
- C#
- .NET MAUI
- XAML
- Visual Studio
- Git & GitHub
